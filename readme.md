# SintONia

## O projeto SintONia foi desenvolvido com HTML, CSS e JavaScript. Primeiro criamos a estrutura do site no HTML, adicionando a navbar, tabela de músicas, formulário e área dos gráficos.

## Depois usamos CSS para estilizar o site, escolhendo cores em tons de roxo, organizando os cards, tabela e formulário para deixar o visual mais bonito e organizado.

## No JavaScript adicionamos as funcionalidades do sistema, como mostrar as músicas na tabela, filtrar por artista ou gênero, adicionar e remover músicas e salvar os dados com LocalStorage.

## Também utilizamos a biblioteca Chart.js para criar gráficos de gêneros musicais e artistas mais ouvidos. Por fim, adicionamos fotos dos artistas para deixar o dashboard mais moderno e visual.
